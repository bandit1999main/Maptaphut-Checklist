🛠️ วิธีสร้าง Firebase Project ฟรีใน 3 นาที
  1. เปิดหน้าเว็บ Firebase Console และล็อกอินด้วยบัญชี Gmail
  2. คลิกปุ่ม **Add project** ตั้งชื่อโครงการ (เช่น *Finance-Tracker*) แล้วดำเนินการต่อจนเสร็จสิ้น
  3. ในหน้าแรกของโครงการ คลิกไอคอน **Web ()** เพื่อสร้างเว็บแอป ตั้งชื่อแอปแล้วกด **Register app**
  4. คุณจะได้รับหน้าต่างโค้ดที่มีชุดข้อความ `const firebaseConfig = { ... }` ให้คัดลอกชุดโค้ดนั้นทั้งหมดมาวางในช่องซ้ายมือ แล้วกดปุ่มบันทึก!
  5. **สำคัญมาก:** ไปที่แถบเมนูด้านซ้ายในหน้าเว็บ Firebase Console:
    • เปิดบริการ **Firestore Database** -> คลิก **Create database** -> เลือกโซนเอเชีย (เช่น `asia-southeast1`) -> กดเริ่มใน **Test Mode** (โหมดทดสอบ เพื่อให้บราวเซอร์เข้าเขียน/อ่านข้อมูลได้ทันที เพื่อให้ระบบบันทึกข้อมูลและไฟล์แนบได้แบบเรียลไทม์)
ตั้งค่าจัดเก็บเอกสารและไฟล์แนบบน Google Drive (5 TB)

เชื่อมต่อกับบัญชี Google Drive ของคุณ เพื่อบันทึกรูปภาพหน้างานและไฟล์เอกสาร PDF ขนาดใหญ่จากแบบตรวจสอบ Checklist ได้โดยตรง ปลดล็อกข้อจำกัด 1MB ของ Firestore ทันที!

สถานะและคู่มือการตั้งค่า
สถานะ: ยังไม่ได้เชื่อมต่อ Google Drive
กรุณากรอก Client ID และ API Key ด้านซ้าย กดปุ่มบันทึก จากนั้นกดปุ่มลงชื่อเข้าใช้เพื่อเปิดสิทธิ์การเข้าถึงโฟลเดอร์
🛠️ วิธีสร้าง Client ID & API Key ฟรีบนคลาวด์ Google
  1. เปิดหน้าเว็บ Google Cloud Console แล้วกดล็อกอิน
  2. เปิดบริการ **API & Services** -> ค้นหาและกด **Enable** บริการ **Google Drive API**
  3. ไปที่เมนู **OAuth consent screen**:
    • เลือกประเภทผู้ใช้เป็น **External** หรือ **Internal** (กรณีบัญชีบริษัทแนะนำ Internal)
    • กรอกชื่อแอปและเพิ่มอีเมลของคุณ จากนั้นกดบันทึก
  4. ไปที่หน้า **Credentials**:
    • คลิก **Create Credentials** -> เลือก **API Key** (คัดลอกมาใส่ช่อง API Key)
    • คลิก **Create Credentials** -> เลือก **OAuth client ID** -> เลือกประเภทเป็น **Web application**
    • ในช่อง **Authorized JavaScript origins** ให้เพิ่ม URL เว็บ Checklist นี้ เช่น `https://bandit1999main.github.io` และ `http://localhost:5500`
  5. กดบันทึกแล้วนำ **Client ID** ยาวๆ ที่ได้มาใส่ในช่องด้านซ้าย
